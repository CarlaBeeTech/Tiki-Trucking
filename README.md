## 📋 Features

- **Responsive design** — looks great on desktop, tablet, and mobile
- **Hero section** with full-screen background and animated zoom effect
- **Services grid** — 6 service categories with hover animations
- **Stats bar** highlighting key business metrics
- **Why Us section** with trust pillars and imagery
- **Haul Request Form** — collects customer order details and sends via email
- **Scroll-reveal animations** throughout the page
- **Fixed navigation** with smooth scrolling

## 🚀 Getting Started

### View locally
1. Clone or download the repo
2. Open `index.html` in any web browser — no server needed

### Deploy to GitHub Pages
1. Push `index.html` to your GitHub repository
2. Go to **Settings → Pages**
3. Set source branch to `main`
4. Your site will be live at `https://yourusername.github.io/repo-name`

## ✏️ Customization

All edits are made inside `index.html`. Use **Ctrl+H** (Find & Replace) in VS Code to quickly swap out placeholder content:

| Placeholder | Replace With |
|---|---|
| `Tiki-Trucking` | Your company name |
| `dispatch@tikitrucking.com` | Your email address |
| `(816) 555-0192` | Your phone number |
| `Kansas City Metro` | Your service area |
| Stats numbers (10+, 500K, 98%) | Your real business stats |

---

## 📬 Form / Order Requests

The haul request form uses a `mailto:` handler — when a customer submits the form, it opens their email client pre-filled with all the order details addressed to your dispatch email. No backend or server required.

To connect a real form backend (for automatic email delivery without requiring the customer's email client), consider free services like:
- [Formspree](https://formspree.io)
- [Web3Forms](https://web3forms.com)
- [EmailJS](https://www.emailjs.com)
